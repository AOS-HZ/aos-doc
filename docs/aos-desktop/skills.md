# 技能

技能页展示 AOS Desktop 在本机发现的所有技能（MCP Server）。你可以在这里确认本机装了哪些技能、它们来自哪里、哪些 AI 工具正在使用它们，以及安全分析的结果。

![AOS Desktop 技能管理](/images/screenshots/skills-management.png)

## 技能卡片里有什么

每个技能会显示：

- **名称和说明**：技能的基本描述
- **本地路径**：技能安装在本机的位置
- **关联工具**：哪些 AI 工具（Claude Code、Cursor 等）正在使用这个技能
- **管理状态**：新发现 / 已安装 / 更新中
- **风险标签**：安全 / 可疑 / 高风险（来自安全分析结果）

技能列表右上角有**刷新按钮**，点击后会重新扫描本机技能目录并更新分析状态。

## 查看技能详情

点击任意技能卡片，会打开一个详情窗口，分两栏显示：

**左栏 - 本地信息：**
- 安装路径和源类型
- 来源引用（如 Git 仓库地址）
- 创建时间、更新时间、最近同步时间

**右栏 - 风险分析：**
- 风险评分和风险等级
- 可疑行为信号
- 可信信号
- 如果来自官方市场，还会显示市场评测结果

详见[技能风险分析](./skill-risk.md)。

## 找不到某个技能

如果你知道本机有某个技能，但列表里没有出现，可以：

1. 点击刷新，等待扫描完成。
2. 在搜索框里按名称或路径搜索。
3. 进入**设置 → 通用 → 扫描路径**，添加这个技能所在的目录。
4. 回到技能页再次刷新。

## 看到不认识的技能

如果列表里出现你不记得安装过的技能，不要只凭名称判断它是否可信。建议：

1. 查看它的安装路径，确认是否是你操作过的位置。
2. 查看关联工具，确认是否有 AI 工具在使用它。
3. 打开详情，查看[技能风险分析](./skill-risk.md)结果。
