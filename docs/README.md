# 认识 AOS Desktop

AOS Desktop 是一款桌面安全工作台，用来帮助你看清 AI 助手在电脑上做了什么、是否出现风险，以及本机安装的技能是否值得信任。

如果你经常使用 AI 编程工具、自动化助手或第三方技能，AOS Desktop 可以帮你把这些活动集中到一个地方查看，避免重要风险被忽略。

## AOS Desktop 适合做什么

### 查看 AI 助手的整体活动

总览页会展示近期活动、用量、成本估算、工具操作和风险数量。你可以用它快速判断今天是否有异常变化。

![AOS Desktop 活动总览](/images/screenshots/runtime-overview.png)

### 复盘一次任务的过程

会话页会把一次任务里的关键步骤串起来。你可以查看 AI 助手调用了哪些工具、哪里出现失败、风险发生在哪一步。

![AOS Desktop 会话画布](/images/screenshots/live-canvas.png)

### 处理安全告警

告警页集中展示需要关注的安全事件。你可以先查看风险说明，再回到对应任务里看上下文，判断它是不是正常操作。

![AOS Desktop 安全告警](/images/screenshots/alerts.png)

### 查看本机技能

技能页会展示本机发现的技能，包括它们的来源、安装位置、关联工具和安全分析结果。

![AOS Desktop 技能管理](/images/screenshots/skills-management.png)

## 你会用到哪些页面

| 页面 | 用来做什么 |
|---|---|
| 总览 | 快速查看整体活动、用量和风险 |
| 会话 | 查看一次任务的完整过程 |
| 告警 | 处理需要关注的安全事件 |
| 技能 | 查看本机技能和安全分析结果 |
| 通知 | 接收风险提醒、会话完成提醒和 Bark 手机推送 |
| 设置 | 调整后台运行、数据保留和其他偏好 |

## 什么时候应该打开 AOS Desktop

- 开始一天工作前，先看有没有新的风险或异常用量。
- AI 助手完成一批任务后，复盘是否有失败或高风险操作。
- 收到风险提醒后，查看告警详情和对应任务。
- 安装或启用新技能前，先查看技能来源和风险分析。
- 觉得某个 AI 工具行为异常时，检查最近的任务记录。
